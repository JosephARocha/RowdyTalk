# RowdyTalk

# Current Goals for RowdyTalk Are:
- Fix Messaging System
- Client GUI
- Server GUI
- Multi-Client Support
- Spell Check
- User-List
- User Profiles and Pictures
- Simple IBM Watson Integration
- Tic Tac Toe 
- Pod Racing...
